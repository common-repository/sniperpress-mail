Add automation to your site by Adding and Removing from email lists in MailChimp and GetResponse on WordPress Hooks (On Register, On Comment). 

== Description ==

Add simple but powerful email automation to your WordPress, and Add/Remove users from multiple Lists in multiple Email Service Providers (ESP).

With SniperPress Mail you can create rules to:

*   Add user to multiple lists from multiple ESP
*   Remove user from multiple lists and multiple ESP
*   Create and combine unlimited rules

Examples:

*   ADD a user to list A in MailChimp and REMOVE it from list B in GetResponse when the user subscribes on WordPress.
*   REMOVE a user to list A in MailChimp and list C in GetResponse when the user leaves a comment on WordPress.

Don't be limited to one Email Service Provider. Use the best service for different lists.
Get an ESP with less features and less expensive for your simple lists and move the buyers to higher end ESP's to valuable lists.

Current ESP's supported by SniperPress Mail:

*	MailChimp
*	GetResponse

== Installation ==

1. Upload `sniperpress-mail.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to SniperMail Press menu

== Frequently Asked Questions ==

= The user must be logged to comment? =

No, if the visitor is not logged in, SniperPress Mail will use the comment email for the automation.
